\# Mutual Fund Transaction \& SIP Management Platform



\## 📌 Overview

A fintech-focused MERN stack application that enables users to perform mutual fund transactions such as buy, redeem, switch, and manage SIPs with secure authentication and scalable backend design.



\## 🚀 Key Features

\- Mutual fund buy, redeem, and switch transactions

\- SIP creation and management

\- SIP auto-retry mechanism for failed payments

\- NAV-based unit calculation logic

\- Secure JWT authentication

\- Role-based access control (Investor/Admin)

\- Transaction-safe REST APIs



\## 🛠 Tech Stack

\- \*\*Frontend:\*\* React.js

\- \*\*Backend:\*\* Node.js, Express.js

\- \*\*Database:\*\* MongoDB

\- \*\*Authentication:\*\* JWT

\- \*\*Architecture:\*\* RESTful APIs
